## Команды консоли:

### Установка зависимостей

```shell
npm install
```

### Сборка проекта для разработки

```shell
npm run start
npm start
```

### Сборка проекта в продакшен

```shell
npm run build
```

### Проверка проекта Линтером

```shell
npm run lint
```

### Запуск превью

```shell
npm run preview
```
